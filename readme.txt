Lightweight plugin inspired by Hello Dolly and the film Ad Astra. Shows random “Humans Manifesto” lines in the shared WordPress admin dashboard.

== Description ==

Humans Manifesto adds a line of text from the "Humans Manifesto" to the top of your WordPress admin dashboard. Each page load displays a different line. The container is shared with the planned "Human Manifesto" plugin, allowing both to appear side by side.

== Installation ==

1. Upload the `humans-manifesto` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. The manifesto will appear at the top of your admin dashboard.
